# Simple-Tasks-API
This API allows you to CRUD tasks.
<h3>Features:</h3>
<ul>
  <li>Create User</li>
  <li>Get-Token</li>
  <li>List Users (only for superuser)</li>
  <li>Create Task</li>
  <li>Get all Tasks</li>
  <li>Update Task</li>
  <li>Delete Task</li>
</ul>
<h3>Technologies Used :</h3>
<ul>
  <li>Django REST framework</li>
  <li>Python 3.9</li>
  <li>MySQL 8.0</li>
  <li>Django 4.1</li>
</ul>
